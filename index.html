Configurar o Visual Studio Code 1. Baixe e instale a versão mais recente do Visual Studio Code para seu sistema operacional. ... 2. Inicie o Visual Studio Code. 3. Na barra de ferramentas à esquerda, clique no ícone de extensões. ... 4. Pesquise o Salesforce Extension Pack (Pacote de extensões do Salesforce) e clique em Install (Instalar).

Você pode Executar console.log clicando no botão "Run" (Executar) ou pressionando Ctrl+Enter . Neste exemplo, deveremos ver "1" no console.
